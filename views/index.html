About Tiramisu

This assignment required students to develop a functional Node.js web server while applying frontend development concepts and Object-Oriented Programming (OOP) principles. Students created a simple HTTP server using the http module, which served different content based on request URLs. They also implemented frontend modules, navigation between pages, and extended a shared PDFGenerator class to generate downloadable PDFs based on various prompts.

Our team developed a webpage called "Tiramisu" with an interactive JavaScript module and a subclass of the PDFGenerator to generate a specific type of document, such as an invoice, course completion certificate, event ticket, or restaurant receipt. The project emphasized collaboration, modular design, and the practical application of web technologies, culminating in a fully functional website where users could preview and download customized PDFs.
